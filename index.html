Seguridad

Esta plataforma no almacena ningun documento, texto ni dato ingresado. El procesamiento se realiza en memoria y el archivo se descarga directo a tu equipo. Los flujos automáticos usan Claude de Anthropic, que no usa datos de API para entrenamiento y los elimina en un maximo de 30 días. Toda la comunicación esta cifrada con TLS 1.3.

Galería de flujos automáticos para funcionarios públicos
Elige un flujo, describe lo que necesitas, y recibe tu documento listo.
Filtrar:
Consideraciones personalizadas Todos los flujos incluyen un campo para darle indicaciones (ej: "considera solo el proceso de matricula", "no incluyas la etapa X").
Precisiones en azul Cuando el flujo tiene dudas o asume algo, lo marca en azul dentro del documento para que lo revises.
Procesos
1
Inventario de Procesos
Crea un inventario desde cero a partir de tus normas, mejora uno existente o desagregalo un nivel más. Incluye campo de consideraciones para guiar el flujo.
Entrega: Word
Procesos
2
Diagrama de Proceso
Adjunta tu ficha de proceso o transcripción, descarga el archivo BPMN e importalo en Bizagi Modeler para tener el flujo listo.
Entrega: BPMN importable
Procesos
3
Ficha de Proceso
Describe el proceso con tus palabras o sube una transcripción de entrevista y obtendras la ficha completa en Google Docs, según la NT 002-2025.
Entrega: Word
Procesos
4
Ficha de Procedimiento
Genera la ficha detallada de cada procedimiento con actividades, responsables, normativa y requisitos, en formato Word.
Entrega: Word
Procesos
5
Mejora de Procesos
Compara tu ficha AS-IS contra la norma sustantiva para identificar vacíos, aplica la metodología ECRS y sugiere oportunidades de automatización con IA.
Entrega: Word
Documentos
Generador de Informes
Pega tus notas o texto desordenado y obtendras un informe técnico en formato MINEDU con antecedentes, análisis, conclusiones y bloques de firma.
